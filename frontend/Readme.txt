# front end

## Shared code

* API
* colors

To create

```
npm init -y
```


## Web

To create

```
npx create-react-app web
cd web
```

Add component

To run

```
npm start
```

## Phone (expo project)

To create

```
cd phone
expo init
```

To run

```
expo start

```
